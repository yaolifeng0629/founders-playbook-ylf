# Founder's Playbook Skill (ylf edition)

[![skills.sh](https://skills.sh/b/yaolifeng0629/founders-playbook-ylf)](https://skills.sh/yaolifeng0629/founders-playbook-ylf)

> AI-native startup founder playbook — a Claude skill based on Anthropic's *The Founder's Playbook: Building an AI-Native Startup* (2026).

## What This Skill Covers

A tactical guide for building startups with Claude as core infrastructure, covering the four-stage lifecycle:

| Stage | Core Question | Primary Claude Tool |
|-------|--------------|---------------------|
| **Idea** | Is this worth building? | Chat + Cowork |
| **MVP** | What should we build first? | Code + Cowork |
| **Launch** | Can this grow repeatably? | All three |
| **Scale** | Can this survive without me? | All three |

## What's Inside

- **Stage-specific guidance** — Goals, exit criteria, challenges, and exercises for each phase
- **Anti-patterns** — Common mistakes and how to avoid them
- **Checklists** — Trackable progress markers for each stage
- **Case studies** — Real AI-native startup stories and resources

## Installation

### Via skills.sh (recommended)

```bash
npx skills add yaolifeng0629/founders-playbook-ylf
```

### Manual Install

Copy this directory into your Claude skills folder:

```bash
# Claude Code
~/.claude/skills/founders-playbook/

# Or use with the Skill tool
/skill founders-playbook
```
