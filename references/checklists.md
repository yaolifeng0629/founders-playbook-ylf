# Founder Checklist by Stage

## Idea Stage
- [ ] Problem statement is specific and testable
- [ ] Claude has argued against the idea and surfaced disconfirming evidence
- [ ] Competitive landscape mapped by tier (direct, indirect, acquirers, adjacent)
- [ ] TAM/SAM/SOM model built and pressure-tested
- [ ] 3 external trends identified (tailwind/headwind assessed)
- [ ] Interview framework drafted and audited by Claude
- [ ] 5+ customer discovery conversations completed
- [ ] Post-interview synthesis: supporting vs. challenging evidence
- [ ] Solution concept stress-tested for critical assumptions
- [ ] Lightweight prototype built (single core interaction only)
- [ ] Prototype tested with 5 people from target profile

## MVP Stage
- [ ] CLAUDE.md architectural context document written
- [ ] MVP scope document: in-scope, out-of-scope, amendment criteria
- [ ] Session template: revisit scope + CLAUDE.md → execute → document decisions
- [ ] Security review completed (auth, data exposure, input validation, dependencies)
- [ ] Measurement framework defined (retention, activation, Day 7/30 targets)
- [ ] False positive definitions documented
- [ ] Sean Ellis test administered
- [ ] Effort test observed (pushing vs. pulling)
- [ ] If 3+ cycles without PMF: pivot diagnostic run

## Launch Stage
- [ ] Architectural audit completed; remediation sequenced
- [ ] Founder attention audit: automate / delegate / keep
- [ ] Security & compliance review
- [ ] Product management OS: sprint cadence, spec template, triage tree, metrics brief
- [ ] Growth channels identified with unit economics (CAC, LTV, payback)
- [ ] Operations running without founder as bottleneck

## Scale Stage
- [ ] Bottleneck map: what stalls when founder is unavailable
- [ ] Enterprise gap analysis for 3 target customers
- [ ] GTM engine: segmentation, messaging, sales playbooks, content pipelines
- [ ] Domain expertise codified into skills and test cases
- [ ] Data flywheel narrative drafted
- [ ] Workflow lock-in audit for top 10 customers
- [ ] Product moat withstands scrutiny: "If a well-funded incumbent copied us today, would users stay?"
